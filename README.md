# Smart Baking – Recipe Contest System

Smart Baking is a PHP + MySQL web application where users submit their recipe name and image, and the admin assigns manual ranks. A leaderboard displays the top recipes.

## Features

- User registration & login
- Submit recipe (name + JPG/JPEG image)
- Admin panel to assign ranks
- Leaderboard with top 3 highlighting
- Animated winner banner on homepage
- MySQL database with `users` and `recipe_submissions` tables
